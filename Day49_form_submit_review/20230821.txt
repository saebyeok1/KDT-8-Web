전송타입
일반폼일 때에는 1번 형식(POST방식에서 제출후에 F12로 네트워크확인시 확인할 수 있음)

GET: 우리가 원하는 데이터를 얻을 때 사용한다고 생각 (받는다)
POST: 우리가 원하는 데이터를 보내면서 받을 때 사용한다고 생각 (보낸다)

input name : 백엔드에서 key 값

label id : 유일한 값 > label for과 id가 동일할 시 둘이 연결됨

body-parser: get방식에선 사용하지 않아도 됨
-> POST로 정보 전송

일반폼: 동기방식으로 전송후엔 그냥 끝임 , 중간에 어떤 일을 처리 할 수 없음
(거의 사용X, 처리 방식만 이해)

readonly: 선택안됨. 백엔드 데이터 전송가능
disabled: 선택안됨. 백엔드 데이터 전송도 안됨
